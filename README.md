# Kampung Rotan

Aplikasi kampung rotan adalah aplikasi yang digunakan untuk menjadi platform untuk para wirausaha dalam produksi rotan dan diikut sertakan
dalam perlombaan innovillage 2021.

## Requirements

- laravel 8
- CSS
- Javascript
- PHP

## Installation

- Clone the repo and `cd` into it
- Run `composer install`
- Rename or copy `.env.example` file to `.env`
- Run `php artisan key:generate`
- Set your database credentials in your `.env` file
